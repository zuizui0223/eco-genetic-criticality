## Editorial rule

The manuscript uses the following labels consistently:

- **Theorem (T):** exact mathematical statement under explicit assumptions.
- **Conditional result (C):** statement that follows only once an ecological or
  life-cycle closure is supplied.
- **Hypothesis (H):** a testable dynamic statement.
- **Finite result (S):** a numerical result for a declared finite closure.

Words such as *prove*, *guarantee*, and *universal* are reserved for T/C claims
with their assumptions stated. All finite numerical results remain Type S.
